# СНТ «Мебельщик»

Сайт садового товарищества у Суражевки, Приморский край.

## Публикация

Сайт автоматически собирается и публикуется в GitHub Pages после каждого
изменения ветки `main`.

Адрес: <https://amichulis1985-create.github.io/snt-mebelshchik/>

## Локальный запуск

```bash
npm ci
npm run dev
```
